Styling Links is a lot more complicated than it looks. It consists of states which means when and how you press the links and which way the link should reacts.

One link can have 4 different states, Those Are:

  1. a:link

    2. When the Link is 1st Visited.

  2. a:visited

    3. When the link has been Already Visited.

  3. a:hover

    4. As your mouse pointer hover over the link.

  4. a:active

    5. As your mouse button is pressed when you're clicking the link.
