Using Image as Pointers In Unordered Listing

To use image as pointers in unordered list we use the property 'list-style-image:url(Images/01.jpg)'

You can also add border into the unordered list by using border property. 'border: 1px solid red;'

By Default there is some padding already exists in the unordered list you can remove it by 'padding:0;'

You can also remove the pointers from the list by using the property 'list-style-type:none;'
