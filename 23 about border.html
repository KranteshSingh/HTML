Learning CSS - About Border

This is tutorial about CSS. XHTML Build the structure of the website like adding text, images etc. CSS does the designing part of the website like font size font color image style etc etc. In the header section of the website we wrote the style statement to add CSS property to our paragraphs. XHTML is white space insensitive.

Border is the area that outlines the padding. Border has 3 properties to play with, These Are as follows:

  1. border-color:red;
  2. border-width:3px;
  3. border-style:dashed;

To change some part of the border you can do it like below:

  • border-top-color:red;
  • border-top-style:dotted;
  • border-top-width:3px;
