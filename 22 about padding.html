Learning CSS - About Padding

This is tutorial about CSS. XHTML Build the structure of the website like adding text, images etc. CSS does the designing part of the website like font size font color image style etc etc. In the header section of the website we wrote the style statement to add CSS property to our paragraphs. XHTML is white space insensitive.

Padding is used to position elements in the web page. Every element, technically consist of 4 parts These Are : 1.Content 2.Padding 3.Border 4.Margin

Padding is the space between the content and its Border. To Specify padding in 4 direction i.e., TOP-RIGHT-BOTTOM-LEFT. You can assign them like below.

  1. padding-top:10px;
  2. padding-right:20px;
  3. padding-bottom:30px;
  4. padding-left:40px;
